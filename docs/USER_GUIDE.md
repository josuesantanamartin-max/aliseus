## ðŸ› ï¸ Solución de Problemas

### La app no carga

1. Verifica tu conexión a internet
2. Limpia caché del navegador
3. Intenta en modo incógnito
4. Actualiza el navegador
5. Contacta soporte si persiste

### No puedo importar CSV

1. Verifica que el archivo sea .csv
2. Asegúrate de que tenga columnas de Fecha, Descripción y Monto
3. Revisa que las fechas estén en formato DD/MM/YYYY
4. Elimina caracteres especiales de los encabezados

### Las recetas no se generan

1. Verifica tu conexión
2. Intenta con una descripción más simple
3. Revisa que tengas créditos de IA disponibles (plan Familia)
4. Contacta soporte si el error persiste

### Los backups no se crean

1. Verifica que tengas espacio en localStorage (~10MB)
2. Limpia backups antiguos
3. Intenta en modo incógnito para descartar extensiones
4. Revisa la consola del navegador (F12) para errores

### No recibo notificaciones

1. Ve a Configuración del navegador
2. Permite notificaciones para Aliseus
3. Verifica configuración de notificaciones en la app
4. Revisa que no estés en modo No Molestar
